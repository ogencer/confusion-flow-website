What is it
You know that thing where you open a Jupyter notebook, write the same boilerplate, then spend 40 minutes wrangling merge keys and column renames?

Confusion Flow replaces that with a canvas. Each operation is a node. Each connection is a data flow. You configure with dropdowns instead of docs. And underneath, it writes the Python code for you — real code you can read, not some proprietary blob. 
[ Read CSV ] ───────────> [ Filter ] ──────> [ Group By ]
  passengers.csv            age > 25              by: class          │
                                                                      │
                            [ Write CSV ] <──────── [ Sort ] <──┘
                              output.csv              desc: mean_fare

This is what a workflow looks like. Five nodes, four edges, zero lines of code written by hand. Hit run and each node executes in order. Click any node to see its output.

What you get

40+ transforms

Filter, join, group, pivot, melt, rename, formula, sample, unique, fill NA, type cast — all the operations you actually use, as drag-and-drop nodes.

Bundled Python 3.13

Ships with a full Python runtime. No install, no PATH headaches, no "works on my machine." Missing a package? It auto-installs via uv in seconds.

Real code output

Every node generates Python code you can inspect. Not pseudocode. Not a DSL. Actual Python that runs in a Jupyter kernel inside the app.

Run all or run one

Execute the full pipeline, run up to a specific node, or right-click and run just one. Data preview shows tables, scalars, and errors inline.

Extensible libraries

Import community tool libraries or write your own. One JSON file defines a new set of nodes with custom code generation.

MCP server built in

Connect to Claude Desktop or Claude Code. Your AI can read your pipelines, add nodes, execute workflows, and preview data — all through the built-in MCP server.
